RFCs

  • [disposition: merge] Allow panicking in constants.
  • [disposition: merge] Allow loop in constant evaluation.
  • [disposition: merge] Introduce #[do_not_recommend] to control errors for trait impls.
  • [disposition: merge] Tuple struct construction with Self(v1, v2, ..).
  • [disposition: merge] Update RFC 0430 to allow underscores between numbers in CamelCase names.
  • [disposition: merge] Add Option::replace to the core library.
  • [disposition: close] Reserve delegate as a keyword in edition 2018.
  • [disposition: close] Reserve f(a = b) in Rust 2018.
